# 이미지 주석 도구 (Image Annotation Tool)

이미지에 박스를 추가하고 편집하여 주석을 달 수 있는 웹 애플리케이션입니다.

## 주요 기능

✅ **이미지 업로드**: 로컬 이미지 파일을 업로드할 수 있습니다.

✅ **박스 생성**: 이미지 위에 주석 박스를 추가할 수 있습니다 (데스크톱: 220x90px, 모바일: 180x75px).

✅ **박스 이동**: 마우스 드래그 또는 터치로 박스를 원하는 위치로 이동할 수 있습니다.

✅ **반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기에서 최적화된 경험을 제공합니다.

✅ **터치 지원**: 모바일 기기에서 터치 제스처로 박스를 조작할 수 있습니다.

✅ **이미지 저장**: 주석이 포함된 이미지를 PNG 파일로 다운로드할 수 있습니다.

✅ **SNS 공유**: 주석 이미지를 소셜 미디어나 다른 앱으로 공유할 수 있습니다 (Web Share API 지원 브라우저).

## 사용 방법

1. **이미지 업로드**
   - "📁 이미지 업로드" 버튼을 클릭하여 이미지 파일을 선택하세요.

2. **박스 추가**
   - "➕ 박스 추가" 버튼을 클릭하면 새로운 주석 박스가 생성됩니다 (220x90px).

3. **박스 이동**
   - 박스를 클릭(또는 터치)하고 드래그하여 위치를 변경할 수 있습니다.
   - 모바일에서는 터치 제스처로 직관적으로 조작할 수 있습니다.

4. **박스 삭제**
   - 삭제할 박스를 클릭하여 선택한 후 "🗑️ 선택된 박스 삭제" 버튼을 클릭하세요.

5. **저장 및 공유**
   - "💾 이미지 저장" 버튼으로 주석이 포함된 이미지를 다운로드할 수 있습니다.
   - "🔗 공유하기" 버튼으로 이미지를 다른 앱이나 SNS로 공유할 수 있습니다.

## 실행 방법

### 방법 1: 직접 열기
브라우저에서 `index.html` 파일을 직접 열어서 사용할 수 있습니다.

### 방법 2: Python HTTP 서버 사용
```bash
# Python 3
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

### 방법 3: Node.js HTTP 서버 사용
```bash
# http-server 설치 (전역)
npm install -g http-server

# 서버 실행
http-server -p 8000

# 브라우저에서 http://localhost:8000 접속
```

## 박스 스타일

### 데스크톱 (480px 이상)
- 크기: 220px x 90px
- 색상: 검회색 (rgba(64, 64, 64, 0.85))
- 모서리: 12px 둥근 모서리

### 모바일 (480px 이하)
- 크기: 180px x 75px (자동 조정)
- 색상: 검회색 (rgba(64, 64, 64, 0.85))
- 모서리: 12px 둥근 모서리

## 반응형 기능

- **데스크톱**: 마우스 드래그로 박스 이동
- **태블릿**: 터치 및 마우스 모두 지원
- **모바일**: 터치 제스처로 박스 이동, 최적화된 버튼 크기 (최소 48px 터치 타겟)
- **자동 조정**: 화면 회전 및 크기 변경 시 레이아웃 자동 재조정
